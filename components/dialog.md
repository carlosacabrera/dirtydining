# Dialog

Description

## Usage

```
<Dialog
    title='Hello, Goodbye'
    onOpen={() => alert('I say hello')}
    onClose={() => alert('You say goodbye')}
>
    <Text>Hello, hello...</Text>
</Dialog>
```

## Usage \(without title\)

```
<Dialog
    onOpen={() => alert('I say hello')}
    onClose={() => alert('You say goodbye')}
>
    <Text>Hello, hello...</Text>
</Dialog>
```
